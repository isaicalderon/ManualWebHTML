Manual Aplicaciones Web / Oracle

  1. Para trabajar con la base de datos, Oracle SQL Developer, se puede utilizar otro manejador de base de datos.

    El instalador se encuentra en la carpeta Oracle SQL Developer dentro del manual. Primero debemos instalar el que viene en la carpeta OracleXE/DISK1.

  2. Hecho esto, debemos de ubicarnos en la carpeta sqldeveloper-4.1.0.19.07-no-jre, abrimos sqldeveloper-4.1.0.19.07-no-jre\sqldeveloper\sqldeveloper.exe

  3. Primero tenemos que configurar una conexión a una base de datos, en nuestro caso utilizaremos la siguiente configuración:
    Nombre del host: ec2-13-58-25-192.us-east-2.compute.amazonaws.com (El host puede cambiar, pregunta al maestro si falla).
    Usuario: CAPACITACION
    Contraseña: CAPACITACION#05032019

    Damos en probar conexión y en Estado: debe de decir “Correcto”. Por ultimo le das en Guardar y Conectar.
